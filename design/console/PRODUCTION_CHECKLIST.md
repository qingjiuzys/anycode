# 818Cloud Console — 生产验收清单

> 上线前验收。套餐仅 Free / Pro / Team，付费仅 monthly / yearly。

---

## 1. 品牌与视觉

| # | 检查项 | 通过标准 |
|---|--------|----------|
| V-01 | 暗色 Console | `/account` 区域暗夜电紫毛玻璃风格 |
| V-02 | 主色 | CTA、进度条 `#7B5CFF` / `--accent` |
| V-03 | 无 mock 占位 | 无圆点按钮、无 test 文案 |

---

## 2. 信息架构

| # | 检查项 | 通过标准 |
|---|--------|----------|
| IA-01 | 默认总览 | `/account` 默认 `section=overview` |
| IA-02 | 四问首屏 | 套餐、账期、token 用量、下一步操作可见 |
| IA-03 | 侧栏额度卡 | 套餐名 + token 进度，点击进用量 |
| IA-04 | 无 Cloud 5h | 全站无「5 小时」「1000 次/窗口」文案 |

---

## 3. 额度与用量

| # | 检查项 | 通过标准 |
|---|--------|----------|
| Q-01 | 账期 token | `tokenUsed / tokenLimit` + 剩余天数 |
| Q-02 | 无滚动窗口 | 无 `calls_remaining`、`quota_resets_at` UI |
| Q-03 | 80% 警告 | 达 80% 显示升级提示 |
| Q-04 | 用尽 | 明确账期结束/升级 CTA |
| Q-05 | 导出 | CSV 可下载 |

## 测试场景

### A：Free → Pro（dev mock）

1. 总览显示 Free + 账期
2. 套餐页选 Pro + 月付 → dev 环境可 mock 升级
3. 总览 token 上限更新

### B：账期额度警告

1. token 使用 ≥80%
2. 用量页升级卡片 + 总览进度条变色

### C：非 Team 访问团队

1. 显示升级引导，成员表不报错
