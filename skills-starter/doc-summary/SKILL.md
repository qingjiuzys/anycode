# doc-summary

> **中文**：批量总结 Markdown、文本或 PDF 类附件。  
> **English**: Summarize Markdown, text, or PDF-like attachments in batch.

Use **Glob** + **FileRead** (and **WebFetch** for URLs) to read documents.
Produce concise summaries: one paragraph per file plus key bullets.
Combine into a single Markdown report when asked.
