# daily-brief

> **中文**：抓取新闻/RSS 主题并生成 Markdown 日报。  
> **English**: Fetch news or RSS topics and produce a Markdown daily brief.

Use **WebSearch** and **WebFetch** to gather recent items on topics the user names.
Output a Markdown brief with: headline, 3–5 bullet summaries, and source links.
Do not auto-post to social media; deliver as a file or chat reply only.
