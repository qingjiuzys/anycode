# content-repurpose

> **中文**：将长内容改写为短文或 alternate 格式（仅草稿）。  
> **English**: Repurpose long content into shorter posts or alternate formats (draft only).

Rewrite source material into the target format the user requests (e.g. short post, outline).
**Do not publish** to external platforms without explicit user action.
Output drafts as Markdown files or chat text only.
