# weekly-report

> **中文**：从笔记、Git 活动或用户要点汇编周报。  
> **English**: Compile a weekly work summary from notes, git activity, or user-provided bullets.

Structure: accomplishments, in-progress, blockers, next week plan.
Use **Glob**/**Grep**/**FileRead** on the project when the user points at a workspace.
Pair well with **CronCreate** for scheduled Friday summaries.
