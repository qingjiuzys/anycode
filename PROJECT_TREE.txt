anycode/
├── Cargo.toml                 # Rust workspace
├── .github/workflows/         # CI, docs publish, release binaries
├── README.md
├── WORKBENCH.md               # Digital Workbench quick-start and operator notes
├── docs/                      # Maintainer docs, ADRs, roadmap, workbench planning/archive
├── docs-site/                 # User-facing VitePress docs site
├── scripts/                   # Install/build/dashboard/eval scripts and fixtures
├── examples/                  # Small runnable examples
│
└── crates/
    ├── core/                  # 领域与 trait：子模块 ids/message/task/llm_types/traits 等，lib.rs 聚合
    ├── agent/                 # AgentRuntime；runtime/*（含 tool_result_injection）为 LLM + 工具循环
    ├── tools/                 # File/Bash/Glob/Grep/MCP/LSP tools and registry
    ├── llm/                   # LLM transports and provider adapters
    ├── security/              # SecurityLayer、审批、策略
    ├── channels/              # Channel abstractions
    ├── memory/                # Memory store and embedding pipeline
    ├── onboard/               # 配置向导等
    ├── locale/                # Fluent bundles and localization support
    ├── dashboard/             # Digital Workbench backend; db/store/{open,*} + API handlers
    ├── dashboard-ui/          # React/Vite workbench UI embedded into release builds
    └── cli/                   # anycode binary: composition root
        ├── app_config/        # schema/{types,validation}, prompts, wizards, load/onboard
        ├── channels/          # WeChat / Telegram / Discord bridges
        ├── commands/dispatch/ # CLI dispatch (channel_cmds, ops, task_cmds)
        ├── cli_args/          # Clap args and subcommands
        ├── tasks/             # run / REPL / workflow execution
        ├── repl/              # stream ratatui REPL
        └── term/              # transcript, styles, session persist

说明：`docs/` 是维护者文档与 ADR；面向用户的文档以 `docs-site/` 为准。`crates/dashboard-ui/` 是 Node/Vite 项目，不是 Cargo workspace member，但由 `crates/dashboard/build.rs` 和 `scripts/build-dashboard-ui.sh` 嵌入发布产物。
