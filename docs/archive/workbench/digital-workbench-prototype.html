生产级 SQLite 存储模型

projects

项目根、业务目标、自动化等级、企业策略、长期状态。

sessions

项目下的每次会话、运行、goal、workflow 执行记录。

agents

跨会话 agent 身份、角色、模型、能力、成本与绩效。

project_events

结构化事件流：turn、tool、gate、artifact、quota、handoff。

artifacts

项目资产：文件、报告、测试结果、截图、交付包、PR。

gates

项目级验收门禁：analyze、test、browser、security、review。

automation_policies

自动化策略：失败重试、换模型、审批、交付标准。

工作台配置

这些配置会保存到浏览器 localStorage。后续接入 SQLite 后会同步到用户配置表。
